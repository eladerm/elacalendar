# Usuarios y Contraseñas de Ejemplo

Aquí tienes la lista de usuarios de ejemplo que se crean para la aplicación. Puedes usar el **nombre de usuario**, el **correo electrónico** (`nombre@example.com`) o el **No. de Trabajador** para iniciar sesión.

La contraseña para **todos** los usuarios es: `12345*`

---

## Administradores Generales
Tienen acceso a todas las funciones.

| Nombre   | Usuario/Correo            | No. de Trabajador |
|----------|---------------------------|-------------------|
| Martin   | `martin` / `martin@example.com` | `1001`            |
| Estefy   | `estefy` / `estefy@example.com` | `1002`            |

---

## Administradores de Sucursal
Tienen acceso a la mayoría de las funciones, pero con algunas restricciones.

| Nombre   | Usuario/Correo                | No. de Trabajador |
|----------|-------------------------------|-------------------|
| Celine   | `celine` / `celine@example.com`   | `2001`            |
| Veronica | `veronica` / `veronica@example.com` | `2002`            |
