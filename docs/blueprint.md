# **App Name**: FirebaseNotes

## Core Features:

- Add Note: Allows users to add new notes with a title and content.
- Live Note Display: Fetches and displays a live list of notes. No Firebase.
- Edit Note: Enables users to edit existing notes and update the changes.
- Delete Note: Allows users to delete notes. No Firebase.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to reflect calmness and learning.
- Background color: Clean white (#F0F8FF) for readability and focus.
- Accent color: Dark gray (#4A5568) for text and interactive elements.
- Body and headline font: 'PT Sans' sans-serif for a modern look and feel.
- Clean and minimal layout to focus on content. Notes are displayed in a clear, organized manner.
- Subtle transitions when adding, editing, or deleting notes for smooth interaction.
